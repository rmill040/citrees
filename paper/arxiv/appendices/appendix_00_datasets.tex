\subsection{Synthetic datasets}

These tables report the number of samples $n$ and features $p$ for each
synthetic design used in the recovery analysis. The Design column gives a short
design label, and the Setting column records the remaining simulation
parameters.

The synthetic datasets were chosen as small controlled examples where the
informative features are known. Each design targets one issue that can matter
for feature ranking: correlated predictors, correlated noise, nonlinear signal,
weak signal, high cardinality noise, redundant features, and an easier and a
harder baseline problem. The same idea is used for the regression datasets,
which also include Friedman~1 and a heteroscedastic design. The parameter
values were set by hand to keep the examples compact and to include one harder
high dimensional, small sample case in each task.
