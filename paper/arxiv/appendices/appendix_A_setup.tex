\section{Setup and assumptions}

\subsection{Fixed-node setup}

Let $(X_i, Y_i)_{i=1}^n$ be the training data. Fix a node $t$ with sample
index set $I_t$ and size $n_t:=|I_t|$. Write
$X_t := (X_i)_{i\in I_t}$ and $Y_t := (Y_i)_{i\in I_t}$ for the data restricted
to that node. Stage~A tests a fixed candidate feature set $F_t$ with
$m_t:=|F_t|$.

The fixed-node theorem only concerns the exhaustive fixed-$B$ Stage~A reference
test: every feature in $F_t$ is tested with the same effective nodewise
per-feature budget $B$, and $F_t$ and $B$ are fixed after conditioning on the
node covariates and label-independent auxiliary randomness.

\subsection{Assumptions and theorem scope}
\label{app:assumptions}

The assumptions below define the inferential target of \cref{sec:theory}. When a
result is described as fixed-node, the node $t$ is held fixed rather than chosen
adaptively from the data.

\paragraph{A0.1 (Fixed node).}\label{assump:fixed-node}
The node $t$ is fixed.

\paragraph{A0.2 (Complete nodewise permutation null).}\label{assump:exchangeability}
Under the null at node $t$, $Y_t$ is exchangeable conditional on $(X_t,U)$.
This is a complete nodewise null, not a featurewise partial-null statement: it
does not cover a null feature when other tested features in $F_t$ remain
associated with $Y_t$.

\paragraph{A0.3 (Label-independent tested feature set).}\label{assump:label-independence}
The tested feature set $F_t$ and effective nodewise budget $B$ are measurable
functions of $(X_t,U)$ and do not depend on the observed labels $Y_t$,
permutation outcomes, or sequential stopping decisions.

\paragraph{A0.4 (Fixed-$B$ computation).}\label{assump:fixed-b}
Each $p_{t,j}$ is computed with exactly $B$ random permutations, with no optional
stopping. The Monte Carlo permutation draws used to compute the $p$-values are
additional randomness, independent of $(X_t,Y_t,U)$, and are not included in
$U$.

\paragraph{A0.5 (Conservative ties).}\label{assump:ties}
The Stage~A $+1$ Monte Carlo $p$-value counts ties conservatively using
$\ind\{T_b \ge T_{\mathrm{obs}}\}$ for right-tail tests. Random tie breaking
among equal best features is label-independent auxiliary randomness and is not
part of the rejection event.
