\section{Conclusion}
\label{sec:conclusion}

We introduced \citrees, a Python library implementing conditional inference
trees and forests with Monte Carlo permutation tests as the inferential
backbone for Stage~A feature screening. The preprint emphasizes finite-sample,
fixed-node (especially root) guarantees in fixed-$B$ mode, and provides a
reproducible benchmark suite for evaluating feature rankings.
