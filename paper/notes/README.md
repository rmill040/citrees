# Writing notes (staging area)

This directory contains internal writing materials that are **not** the
manuscript.

- `notes.md`: long-form draft notes (messy is fine); migrate content into
  `paper/arxiv/` as sections stabilize.
- `figures_plan.md`: mapping from scripts → cached artifacts → paper-facing
  figures/tables → the claim/story each output supports.
- References live in `paper/arxiv/references.bib` (single source of truth for
  citations). If you add a new citation in notes, add the BibTeX entry there.
